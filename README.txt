BA12 Automotive Netlify CMS Setup
=================================

1. Upload the /admin folder to the root of your GitHub repo.
2. Upload /cars and /images folders too.
3. Connect your site to Netlify and enable:
   - Identity -> Enable Identity
   - Identity -> Enable Git Gateway
   - Identity -> Invite Users to access /admin
4. Visit /admin and log in.
5. Listings you add will be saved as Markdown in /cars.
6. Use car-template.html to render vehicle pages.
7. The form.html file allows enquiries to be sent via Formspree.
